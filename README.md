# 🍳 주간 밀프랩 가이드

일주일 단위 밀프랩 식단을 위한 레시피 모음 & 식단 플래너

🔗 **Live**: [jadekim042386.github.io/meal-prep](https://jadekim042386.github.io/meal-prep/)

## 주요 기능

- **98개 레시피** — 한식 / 양식 / 아시안 / 멕시칸 / 지중해식 / 간편식
- **주간 식단표** — 4가지 모드(균형 / 한식 위주 / 고단백 / 초간편)로 자동 생성
- **장보기 리스트** — 식단표 기준 자동 생성, 체크박스 + localStorage 저장
- **레시피 검색** — 요리명, 재료, 태그 검색
- **카테고리 필터** — 탭으로 빠른 필터링

## 기술 스택

- 순수 HTML / CSS / JS (단일 파일, 외부 라이브러리 없음)
- GitHub Pages 배포
- CSS `@property`, `animation-timeline`, `backdrop-filter`
- IntersectionObserver, View Transitions

## 로컬 실행

```bash
open index.html
```
